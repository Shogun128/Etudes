Variables

Types de variables

Il existe deux (2) types de variables:

  • Les variables quantitatives
  • Les variables qualitatives

Types de variables quantitatives

Il existe deux (2) types de variables quantitatives:

  • Variables quantitatives discretes
  • Variables quantitatives continues

Les variables quantitatives discretes n'ont qu'une partie entiere alrs que les variables quantitatives continues ont une partie entiere et une partie decimale.

Types de variables qualitatives

Il existe deux (2) types de variables qualitatives:

  • Variables qualitatives ordinales
  • Variables qualitatives nominales

Les variables qualitatives ordinales peuvent etres ordonnees tandis que les variables qualitatives nominales ne peuvent etre ordonnees.
